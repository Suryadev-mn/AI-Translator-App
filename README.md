# 🌍 AI Translator App

An AI-powered multilingual translation application built with **Python** and **Streamlit**.

# 🌍 AI Translator App

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)](https://ai-translator-app-ht6sylfhdaz3ypfx5csa3f.streamlit.app)

## Features

* 🌐 Automatic language detection
* 🌍 Support for multiple languages
* 🔄 Fast text translation
* 🔊 Text-to-Speech
* 📥 Download translated text
* 📜 Translation history
* 📊 Translation statistics
* 🎨 Modern dark-themed interface

## Technologies Used

* Python
* Streamlit
* deep-translator
* langdetect
* gTTS

## Installation

```bash
git clone <repository-url>
cd AI-Translator-App

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
```
